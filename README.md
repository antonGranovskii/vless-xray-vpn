# vless-xray-vpn


## Подключение к VPN

### Скачать впн клиент. Я рекомендую версию 17.16.4 так как она использовалась для тестов:
```bash
https://github.com/2dust/v2rayN/releases
```

### После установки откройте клиент и в левом верхнем углу нажмите "Configuration" -> "Import Share Links From clipboard"

###  вас должно появиться следующее:

<img width="1190" height="876" alt="image" src="https://github.com/user-attachments/assets/aac8750c-d69b-482f-8770-dd24dec4efdf" />


### После выставите следующие настройки (Обведено Красным на скриншоте сверху)
```bash
- Routing: V4 (Global)
- Enable Tun: ON
- Set system proxy
```

## Отключение от VPN
В меню где мы выставляли "Set system proxy" выбрать  "Clear system proxy". Это сразу же отключит вас от VPN. Можно увидеть это если проверить свой IP.

## Проблемы при подключении
Наиболее вероятная проблема - интернет перестает работать. Вообще. Google перестает отдавать ответы с ошибкой ERR_PROXY_CONNECTION_FAILED в браузере.
### Что делать,
1. Порядок включения настроек важен. Сначала V4 Global, затем Enable Tun и только потом  Set system proxy. Если не уверены, чтобы сбросить подключение просто выберите "Clear system proxy" и выберите нужные настрйоки заново.
2. Поиграться с "Enable Tun". Иногда он не нужен и его можно держать в выключенном состоянии. Если выбрано ON (зеленый тумблер), то весь трафик идет через VPN. Но так не сработало из РФ.
3. Поставить dns сервера на 8.8.8.8 гугловские.
